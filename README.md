
# SightSafe - AI Vision Assistant for Seniors

An AI-powered accessibility solution that helps visually impaired seniors navigate safely through real-time obstacle detection and voice-guided instructions.

## Overview

SightSafe addresses a critical need: **2.2 billion people worldwide have vision impairment**, and falls are the leading cause of injury among seniors. This project provides an intelligent, user-friendly system that detects hazards and provides clear audio guidance for safe navigation.

## Key Features

- **Smart Detection** - Identifies obstacles, stairs, uneven surfaces, and people with 3-level risk assessment
- **Voice Guidance** - Senior-friendly audio instructions with visual feedback
- **History Tracking** - Logs scans for safety monitoring and caregiver review
- **Multiple Interfaces** - Streamlit web app, Node.js CLI, and standalone HTML version

## Tech Stack

**Frontend:** Streamlit, HTML5, CSS3, JavaScript  
**Backend:** Python 3.8+, Node.js  
**Features:** Real-time scanning, session logging, responsive design

## Quick Start

```bash
# Web App
pip install -r requirements.txt
streamlit run app.py

# CLI Tool
node app.js

# HTML - Just open index.html in browser
```
